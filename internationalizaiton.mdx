# Internationalization (i18n)

Internationalization (i18n for short) is configuring a codebase to support multiple languages. I recently spent some time setting up our react-native application to detect and set the language if it's available. Through this, I have been learning some very interesting things that didn't turn out to be very hard in setting it all up.
